Github Pages + Custom Domain Guide for 404 Cafe & Club

note: github requires your 404 page be called 404.html instead of not_found.html

Navigate to your Github repo and go to settings

On the left, click on "Pages"

Under "Branch" select "main". Then hit Save.

Under "Custom Domain" type 404cafe.club and click "Save"

We're done with github now! Now go to porkbun.com

porkbun has a great guide for setting up your domain with github pages, https://kb.porkbun.com/article/64-how-to-connect-your-domain-to-github-pages
